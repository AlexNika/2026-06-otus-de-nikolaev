# Домашнее задание №1 по уроку №2 "Инженер данных. Архитектуры систем обработки данных"

## Цель:
В этом ДЗ вам нужно сделать обзор вакансий Data Engineer по нескольким ресурсам (hh.ru, работа.ру и др.).
Составить список требуемых навыков, продуктов и технологий.


## Описание/Пошаговая инструкция выполнения домашнего задания:
Найти и изучить несколько ресурсов с вакансиями Data Engineer (hh.ru, работа.ру и др.).
Выписать требования к навыкам и опыту работы с продуктами и технологиями.
Обобщить.

## Критерии оценки:
- Приведён список источников вакансий.
